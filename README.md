#joelmercier.io

My personal portfolio built with Jekyll. All the tasks are automated with Gulp. Styles are done with Sass. jQuery is used to interact with the DOM and Barba.js handles the PJAX navigation on the site.

You can see it live [here](https://joelmercier.io)

##Install

###Requirements

To use the site you need to have Ruby (v2 or above), Composer and NodeJS installed on your machine.

###How to

First use composer by using the command `composer install`, then download all the node dependencies by using `npm install`.
You then need to install Jekyll, for this simply use the command `gem install jekyll`
